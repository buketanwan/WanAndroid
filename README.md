# WanAndroid java客户端

## 1.简介
本项目是根据鸿洋大大提供的api，模仿@JsonChao的界面进行编写。<br/>
项目基于Material Design + MVP + Rxjava2 + Retrofit + Glide 框架。

## 2.Tips
欢迎朋友们多多star支持。如果发现任何bug，欢迎提出issue或者邮箱联系sheajin@126.com <br/>

## 4.框架介绍 
* 使用Rxjava2结合Retrofit2进行网络请求。<br/>
* 使用Rxjava2的操作符对事件流进行进行转换、延时、过滤等操作。<br/>
* 整体UI使用Material Design中的Behavior、ToolBar、BottomNavigationView等。<br/>
* 使用FloatingActionButton悬浮按钮,快速滑到顶部。<br/>
* 使用SmartRefreshLayout刷新界面,提升界面美感。<br/>
* 项目中多处使用了炫目的动画及特效。<br/>

* 想了解更多欢迎下载代码查看。。<br/>
